This page will send HTTP requests to the "/hello" route of the Kitura sample server. The server will then process the incoming request to return or change the global "name" variable. This demonstrates Raw routing in Kitura.


  • Send an HTTP GET request to "/hello" to ask for "name". If no name is set the server will default to "World".

  • Send a JSON encoded HTTP POST request to "/hello" to set the "name".
